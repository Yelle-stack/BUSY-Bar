BUSY Bar est un dispositif multi-outil de productivité avec un écran LED pixelisé. Minuteur de concentration avec fonction de blocage des distractions sur votre téléphone et PC. Entièrement personnalisable, open-source et prêt pour la maison connectée.

Outil de productivité

Blocage des distractions, message BUSY personnalisé, minuteur de concentration, synchronisation multiplateforme

Application et intégration

Bibliothèque d'applications, connexion à des logiciels tiers, intégrations avec les événements de calendrier et les appels

Support maison connectée

Connectez-vous à Google Home et Apple Home via le protocole Matter

Pour les développeurs

API HTTP ouverte, SDK open-source, bibliothèque Python / Go / JavaScript, MQTT, sans verrouillage fournisseur
